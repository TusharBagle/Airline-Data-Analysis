# Airline Data Analysis

# ✈️ Airline Data Analysis Dashboard

An advanced Microsoft Excel project that analyzes airline customer reviews and satisfaction using Excel's powerful data analysis and visualization features. The project demonstrates data cleaning, formulas, KPIs, Pivot Tables, dashboards, and business intelligence techniques on a real-world airline review dataset.

---

## 📌 Project Overview

This project transforms raw airline customer review data into meaningful business insights through an interactive Excel dashboard.

The workbook covers the complete data analytics workflow:

- Data Loading
- Data Cleaning
- Formula-based Analysis
- KPI Calculation
- Pivot Tables
- Data Visualization
- Interactive Dashboard
- Documentation

---

## 📂 Workbook Structure


| Worksheet              | Description                            |
|------------------------|----------------------------------------|
| **Load Dataset**       | Raw airline review dataset             |
| **Data Cleaning**      | Cleaned and prepared dataset           |
| **Formulas**           | Excel formula practice using real data |
| **Pivot Tables**       | Business summaries using Pivot Tables  |
| **Data Visualization** | Charts and graphical analysis          |
| **KPI's**              | Key Performance Indicators             |
| **Dashboard**          | Interactive Airline Dashboard          |
| **Documentations**     | Project documentation                  |
